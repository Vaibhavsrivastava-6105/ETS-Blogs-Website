# ETS Blogs Website

This is a comprehensive, full-stack Next.js 15 blog platform built for modern content creation.

## 🚀 Live Website

**Production URL:** [https://ets-blogs.vercel.app](https://ets-blogs.vercel.app)

## 🛠️ Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Database:** MongoDB
- **Editor:** Tiptap (Rich Text Editing)
- **Styling:** CSS
- **Deployment:** Vercel

## 📖 Features
- **Admin Dashboard:** Full dashboard to manage articles, categories, and drafts.
- **Rich Text Editor:** Build beautiful articles with formatting, lists, and code blocks.
- **Auto-saving Drafts:** Never lose your work with background auto-saves.
- **Category Management:** Categorize articles dynamically.
- **Responsive Design:** Completely responsive layout that works on desktop, tablet, and mobile.

## 💻 Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
